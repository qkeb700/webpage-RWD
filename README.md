## 개발환경
- Visual Studio Code

## 프로젝트 개요
- HTML 메인페이지를 Header, Section, Footer 로 나누어 배치
- float 와 ::after 를 기반으로 col 간격을 각각 설정 후 배치
- CSS로 이미지와 박스, 글꼴 등 디자인
- 웹으로 실행했을 때와 모바일로 실행햇을 때 별로 다른 속성값을 지정해줘서 때마다 다르게 보이게 구현
